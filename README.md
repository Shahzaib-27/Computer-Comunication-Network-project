# Hotel Management System Network (Cisco Packet Tracer)

A complete **Hotel Management System Network** designed and implemented in **Cisco Packet Tracer**. This project simulates a real-world hotel network by connecting multiple departments through a secure and efficient infrastructure.

## Features

* 🏨 7 Department-based VLANs (Guest, Office, Reception, Staff, Management, Security, CCTV)
* 🌐 Router-on-a-Stick for Inter-VLAN Routing
* 🔒 Network Address Translation (NAT) for Internet connectivity
* 📡 DHCP Server for automatic IP address assignment
* 🌍 DNS Server for domain name resolution
* 💻 Web Server hosting the hotel website
* 📁 FTP Server for secure file sharing
* 📧 Email Server (SMTP & POP3) for internal communication
* 🔀 Trunk and Access Port configuration
* ✅ Network testing using Ping, DHCP, DNS, HTTP, FTP, and Email services

## Technologies Used

* Cisco Packet Tracer
* VLANs
* Router-on-a-Stick
* NAT
* DHCP
* DNS
* HTTP/Web Server
* FTP Server
* Email Server (SMTP & POP3)

## Project Objective

The goal of this project is to design a secure, scalable, and reliable enterprise network for a hotel. It demonstrates practical networking concepts such as VLAN implementation, routing, switching, server configuration, IP addressing, network security, and troubleshooting.

This project was developed as part of the **Computer Communication Network (CMSL-318)** course and provides hands-on experience in designing and managing enterprise-level networks.
